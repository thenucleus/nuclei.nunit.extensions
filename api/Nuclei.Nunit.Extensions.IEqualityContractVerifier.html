Interface IEqualityContractVerifier

Defines the interface for tests that need to verify that the equality contract is implemented correctly.

Namespace:Nuclei.Nunit.Extensions
Assembly:Nuclei.Nunit.Extensions.dll
Syntax
public interface IEqualityContractVerifier
Remarks

The interface is defined so that the EqualityContractVerifierTest can call methods on a non-generic object, while the user can provide a generic class.

This code is based on, but not exactly the same as, the code of the hashcode contract verifier in the MbUnit project which is licensed under the Apache License 2.0. More information can be found at: https://code.google.com/p/mb-unit/
